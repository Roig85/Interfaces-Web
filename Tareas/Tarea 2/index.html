Tarea 2

Objetivos

El objetivo de esta tarea es que el o la estudiante comience a integrar los temas del curso y practique todo como un todo.

El estudiante deberá desarrollar una página para un restaurante imaginario (pueden copiar una página de un restaurante existente).

Evaluación

Rubro Porcentaje
Responsive 20%
Selectores de CSS reutilizables 10%
HTML semántico 30%
Mostrar por lo menos 8 productos 10%
Mostrar una navegación aunque no sea utilizable 10%
Mostrar un footer 10%
Mostrar un banner de algún tipo 10%
