Tarea 0: Tributo

Objetivos

El objetivo de esta tarea es que el o la estudiante tenga una primera interacción con HTML, CSS y CSS Box Model.

El o la estudiante deberá crear una página "tributo" a una persona, personaje, grupo, etc. que admire, le apasione, o le guste. Deberá tener como mínimo:

  1. Un encabezado con el nombre de la persona a la que se le hace el tributo.
  2. Un texto con una breve descripción de qué hace la persona, o por qué es famoso.
  3. Una imagen, con un tamaño apropiado y un borde no cuadrado. Las dimensiones quedan a criterio del estudiante. La imagen debe tener un caption o descripción. La imagen debe tener un alt text.
  4. Una línea de tiempo (puede ser una lista) de logros, publicaciones, películas, etc. de la persona. No tienen que estar todas.
  5. Un pie de página con la información del autor de la página, es decir, el o la estudiante y el año (2021).

Evaluación

Rubro Porcentaje
Encabezado 10%
Descripción breve 10%
Que se vea una imagen 10%
Que la imagen tenga un alt 10%
Que se modifique el tamaño de la imagen y el radio del borde 10%
Línea del tiempo 30%
Pie de página 20%
